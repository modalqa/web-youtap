# Setup

## Install Node Version Manager

<https://github.com/nvm-sh/nvm>

For unix users: <https://github.com/nodenv/nodenv>

Then install node version 14.16.0

## Install pnpm

<https://pnpm.js.org/>

## Install packages

`pnpm install`

## Run on Cypress Dashboard

`pnpm cy-open`

## Run on Headless

`pnpm cy-cons`
